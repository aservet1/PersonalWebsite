Projects

Energy Monitoring Web App
Tools Used: Java, C, Java Native Interface React.js, Something Something about HTTP
Objective: Create a way of remotely querying a machine's energy consumption via a web app.

Method: I already had an energy querying Java library that I had worked on for my undergraduate research project. I then built a Java HTTP server that takes GET requests and returns the results in JSON format. The server interacts with my React.js frontend, a Single Page Application which can render different metrics for querying the energy of the remote machine. It sends an Axios GET request to the backend, picks up the JSON energy readings, and displays them to the user. This is a work in progress, so there are currently only two working features: A snapshot of the system's energy counters (an accumulating total of how many joules the machine consumed) and how much energy the machine consumes in the next X milliseconds. I am working on other features, like longer term average energy usage over time or computer generated graphs of energy activity.

Unix Shell Todo List Application
Tools Used: Shell Scripting, Unix tools (sed, grep, etc), POSIX Regular Expressions
Objective: To make a command line todo list manager. I had too many todo.txt files for different topics and it got cumbersome to manage and view them all over my file system so I developed some scripts to make it easier to view them, and it evolved into a full application. This project helped me learn about the Shell as a newbie Linux user.

Method: I have a list of text files stored in one of this project's folders and a collection of Bash scripts used to do different operations on them. One master script, which I can run from any directory in the command line, reads the first command line argument and then passes the other arguments another script that facilitates the view, create, delete, and update operations allowed. The tool uses POSIX regular expressions to parse command line arguments and Unix tools like sed and grep when doing slightly more advanced viewing operations, such as viewing all lines with a list marked with a "priority" tag.
